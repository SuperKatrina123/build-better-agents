# 做好一个 Agent

> 一套面向有 Agent 基础开发者的系统化知识库。
> 从上下文工程到生产化，理论结合实战，每章围绕同一个贯穿案例落地。

---

## 这是什么

不是 API 教程，不是框架使用手册。

是一套帮你回答这些问题的知识体系：

- 为什么我的 Agent 做着做着就"迷失"了？
- 上下文该怎么管？工具该怎么设计？
- 如何让 Agent 跨会话不失忆？
- 长任务崩溃了怎么断点续传？
- 什么时候需要多 Agent，怎么设计？
- 怎么评测 Agent 做得好不好？
- 如何从原型走向生产？

---

## 贯穿案例

整套知识库围绕一个真实场景展开：

**功能追齐 Agent** — 将 15 个 H5 页面的功能自动对齐到 App 最新版本。

每章的概念都在这个 Agent 上落地，读完即可上手。
